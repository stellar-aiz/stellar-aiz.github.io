Client Management

Centralized Client Database

Centrally manage client information and visualize transaction history by project through automatic linking. By tracking sales trends and transaction history for each client, you can support strategic sales activities.

Key Features

  • Register and edit client information (company name, contact person, contact details)
  • Track transaction history through automatic project linking
  • Aggregate sales performance and profitability by client
  • Quick client search with search and filtering

Use Cases

Instantly answer questions like "What is the total transaction amount with this client?" or "Which clients generate the most revenue?" Use it for sales strategy planning and client portfolio analysis.

Talent Management

Team Member Database

Centrally manage information about team members (talents) participating in projects. Register billing rates, costs, and affiliated companies, with automatic rate population when creating estimates. This forms the foundation for accurate profitability calculations.

Key Features

  • Register and edit member information (name, employment type, affiliated company)
  • Set weekly billing rates and costs
  • Automatic rate population during estimate creation (master data integration)
  • Visualize utilization status and revenue contribution by talent

Use Cases

Simply select a talent when creating an estimate, and registered rates and costs are automatically populated. This prevents manual entry errors and enables accurate profit margin calculations. You can also analyze revenue contribution by member and optimize resource allocation.

Role Management

Job Role Definitions

Define job titles and roles used in projects. Register roles such as Consultant, Engineer, Designer, etc., and apply them to estimate line items to clearly manage hours and rates by job category.

Key Features

  • Register job titles and roles (Consultant, PM, Engineer, etc.)
  • Apply to estimate line items (role-based input when talent is not selected)
  • Set standard rates by role
  • Customize to match your organization's job structure

Use Cases

Even when assigned members are not yet determined, create estimates on a role basis such as "Senior Consultant x 2." Talents can be assigned later, enabling flexible estimate management.

Project Management

Engagement Tracking

Centrally manage basic information for consulting engagements. Register client associations, project duration, status, and more to get a complete overview of each engagement. This serves as the key master data that drives estimates, billing, and profitability analysis.

Key Features

  • Register and edit project information (project name, client, duration)
  • Status management (Proposal, In Progress, Completed, etc.)
  • Automatic linking with client master data
  • Search and filter in project list

Use Cases

When you register a project at the start of a new engagement, all related data including estimates, billing, and timesheets are automatically linked. Track profitability in real-time throughout the entire project lifecycle.

Estimate Management

Proposal & Pricing

Create and manage project estimates. Equipped with weekly hour allocation, version control, and approval workflows. Automatic rate population through talent master integration enables accurate and efficient estimate creation.

Key Features

  • Line item-based estimate creation (talent/role selection)
  • Weekly utilization (allocation) settings
  • Version control (maintain history with v1, v2...)
  • Approval workflow (Draft -> Pending Approval -> Approved)
  • Automatic calculation of cost, revenue, and gross profit

Use Cases

Create estimates for client proposals and update versions as negotiations progress. Approved estimates are locked for editing and used as finalized data for billing and profitability calculations. Since you can confirm profit margins at the estimate stage, you can make proposals with profitability in mind.

Assignment Management

Resource Allocation

Visualize member assignment status across projects. Automatically aggregate planned workloads from approved estimates and see resource availability at a glance. Supports optimal resource allocation and utilization management.

Key Features

  • Utilization status list by talent
  • Weekly and monthly allocation views
  • Alert display when utilization exceeds 100%
  • Automatic integration with approved estimates

Use Cases

Instantly answer questions like "Who is available next month?" or "Is this member overallocated?" Use it for resource adjustment during project staffing or for deciding whether to take on new engagements.

Billing Management

Invoice Scheduling

Manage billing schedules based on approved estimates. View monthly and annual billing plans in a list to prevent missed invoices. Seamless data integration from estimates to billing enables accurate revenue management.

Key Features

  • Monthly and annual billing schedule list
  • Automatic billing data generation from approved estimates
  • Billing amount aggregation by client and project
  • Flexible billing period settings

Use Cases

At month-end billing time, view all projects that need to be invoiced in a list. Since it's linked to estimate data, billing calculation errors and missed invoices are prevented. Can also be used for annual revenue forecasting.

Timesheet Entry

Time Tracking

Timesheet functionality for members to record daily working hours. Enter hours by project and date, and track variances from estimates in real-time. Forms the foundation for accurate cost calculation and hour management.

Key Features

  • Daily and project-based working hour entry
  • Weekly and monthly aggregation display
  • Approval request submission functionality
  • Variance display against estimated hours

Use Cases

Members enter working hours at the end of each day. Submit approval requests at the end of the week or month. Managers can detect discrepancies between actuals and estimates early, monitoring project profitability in real-time.

Timesheet Approval

Manager Review

Manager functionality to approve or reject submitted timesheets. Verify the reasonableness of hours and approve to finalize actual data. Strengthens governance over hour management.

Key Features

  • List display of pending approval timesheets
  • One-click approve and reject operations
  • Filter by member and period
  • Lock approved actuals (prevent editing)

Use Cases

During weekly or monthly close processing, managers review submitted timesheets. If there are questions, reject and request corrections from the member. Approve if there are no issues. Approved data is reflected in profitability calculations, enabling accurate P&L analysis.

P&L Management

Profit & Loss Analytics

Dashboard for analyzing profitability by project and member in real-time. Automatically aggregate revenue, cost, and gross profit with comparative analysis by specified periods. Provides the data needed for management decisions instantly.

Key Features

  • Weekly P&L summary (revenue, labor cost, expenses, gross profit)
  • Profitability contribution analysis by project
  • Revenue contribution analysis by talent
  • Flexible analysis with period filters
  • Visualization with graphs and charts

Use Cases

Report business profitability status immediately at management meetings or weekly meetings. Answer questions like "What is this month's gross margin?" or "Which projects are contributing to profit?" with real-time data.

Stellar AI

AI-Powered Analytics

A copilot feature that analyzes and responds to profitability data simply by asking questions in natural language to the AI agent. Automate complex report creation and aggregation tasks, accelerating data-driven decision making.

Key Features

  • Natural language P&L data queries
  • Profitability analysis by project and talent
  • Context understanding integrated with period filters
  • Deep-dive analysis through multi-turn conversations

Example Queries

"What is this month's gross margin?"
"Which project is contributing the most to profit?"
"What is John's revenue contribution?"

Use Cases

Executives and managers can instantly retrieve information they need without navigating report screens. Simply ask "What is the profit margin for this engagement?" and the AI analyzes the data and responds. Lower the barrier to data analysis and enable everyone to make data-driven decisions.
