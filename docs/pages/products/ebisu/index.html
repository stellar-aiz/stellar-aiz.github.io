Features

主な機能

ステラ・エコノミクスが提供する主要な機能をご紹介します

顧客管理

クライアント情報を一元管理。プロジェクトとの紐付けで案件別の取引実績を可視化

タレント管理

メンバーの単価・原価・所属会社を管理。見積作成時に自動で単価を反映

ロール管理

コンサルタント、エンジニア等の職種を定義。見積の明細行に適用

プロジェクト管理

案件の基本情報を登録。顧客・期間・ステータスを一覧で把握

見積管理

バージョン管理付きの見積作成。週次の工数配分と承認ワークフローを搭載

稼働管理

メンバーのアサイン状況を可視化。稼働率とリソース配分を最適化

請求管理

承認済み見積に基づく請求スケジュール。月次・年次で請求予定を管理

実績入力

タイムシート入力。日別の稼働時間を記録し、見積との差異を把握

実績承認

提出されたタイムシートを承認・差戻し。工数管理のガバナンスを強化

管理PL

プロジェクト別・メンバー別の収益分析。売上・原価・粗利をリアルタイム表示

Stellar AI

AIエージェントに質問するだけで収益データを分析。自然言語で洞察を取得
